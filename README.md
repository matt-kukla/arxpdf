# arxpdf
A shell script for downloading arXiv preprints as PDFs.

## Prerequisites
* ```curl```

## Usage
```arxpdf [arXiv id]...``` to download a PDF of [arXiv id]

```arxpdf [arXiv id]vn...``` to download the nth version of [arXiv id]
